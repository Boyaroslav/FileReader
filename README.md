This script can display text files in terminal beautiful.
<br>
to install:
<br>
    git clone https://github.com/Boyaroslav/FileReader
<br>
    cd FileReader
<br>
    sudo make install
<br>
to use:
<br>
    fr [filename]
<br>
        -r - rainbow colors
<br>
        -t [Speed] - speed of printing
<br>
        -f - instant printing
